## 数组

### 交换变量

```js
let a='b',
let b='a'
[a,b]=[b,a];
```

### 数组赋值

```js
const arr=["鸡同学"，"鸭同学"，"蛋同学"];
const [num1,num2]=arr;
log(num1,num2);
//鸡同学 鸭同学
```

### 对象

```js
const obj ={
id:1",
name_:'蛋老师'，
age:18,
gender:'男'，
email:709394@qq.com',
}
const {name_,email }=obj;
console.log(name_)//蛋老师
```

### 重构赋值

```js

id:1',
name:.'蛋老师'，
age:18,
gender:'男'，
email:709394@qq.com',
}

```
